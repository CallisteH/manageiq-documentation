= Cloud Networks

{product-title} allows you to create and manage network objects for cloud provider software-defined networking components. Network objects include *Networks*, *Floating IPs*, *Subnets*, *Security Groups*. 

{product-title_short} enables configuration and administration for the software-defined networking component of Red Hat OpenStack Platform. The virtual network infrastructure enables connectivity between instances and the physical external network.

This section describes common cloud network administration tasks, such as adding and removing subnets and routers to suit your Red Hat OpenStack Platform providers.
